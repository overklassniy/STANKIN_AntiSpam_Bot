# СТАНКИН Анти-спам — Ассеты

Дизайн-ассеты проекта [STANKIN AntiSpam Bot](https://github.com/overklassniy/STANKIN_AntiSpam_Bot).

## Структура

```
├── logos/                       # Логотипы (Adobe Illustrator)
│   ├── stankin-logo.ai          # Логотип СТАНКИН
│   └── stankin_max.ai           # Расширенный логотип
│
└── emojis/                      # Эмодзи для уведомлений бота
    ├── 94.png                   # Порог 94%
    ├── 98.png                   # Порог 98%
    ├── NS.png                   # Not Spam
    └── mute.png                 # Mute
```

## Использование

```bash
git checkout assets
```

Логотипы используются для брендинга. Эмодзи ранее использовались в уведомлениях бота.
