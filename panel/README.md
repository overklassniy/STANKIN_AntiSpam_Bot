# Веб-панель

FastAPI-приложение для управления системой модерации: просмотр журнала спама, ограниченных пользователей, настройка параметров, управление чатами.

## Структура

```
panel/
├── app.py               # Создание FastAPI-приложения, middleware, lifespan
├── schemas.py           # Pydantic-модели для API-ответов
├── routes/              # Маршруты
│   ├── api.py           # REST API (/api/v1/): JSON-эндпоинты для фронтенда
│   ├── auth.py          # Аутентификация: session-based, проверка прав
│   ├── settings.py      # Страница настроек (HTML)
│   ├── spam.py          # Страница спам-журнала (HTML)
│   └── muted.py         # Страница ограниченных пользователей (HTML)
├── src/                 # Исходный код фронтенда
│   ├── ts/              # TypeScript: API-клиент, UI-логика, пагинация
│   ├── scss/            # SCSS-стили: переменные, компоненты, layout
│   └── css/             # Tailwind CSS entry point
├── static/              # Скомпилированные статические ресурсы
│   ├── js/              # Скомпилированный JavaScript
│   ├── styles/          # Скомпилированные CSS-стили
│   ├── fonts/           # Шрифты
│   └── images/          # Изображения
└── templates/           # Jinja2-шаблоны HTML-страниц
    ├── base.html        # Базовый шаблон
    ├── login.html       # Страница входа
    ├── main.html        # Главная страница
    ├── settings.html    # Страница настроек
    ├── muted.html       # Страница ограниченных
    ├── 404.html         # Страница 404
    ├── 500.html         # Страница 500
    └── exception.html   # Шаблон ошибок
```

## REST API

REST API находится под префиксом `/api/v1/` и возвращает JSON-данные для клиентского TypeScript. Документация API доступна через Scalar по адресу `/scalar` при запущенном приложении.

Подробное описание эндпоинтов — в [.docs/api.md](../.docs/api.md).

## Аутентификация

Session-based аутентификация с проверкой доступа к чатам:

- Пароли хранятся в формате scrypt
- Сессия хранится в cookie с настраиваемым временем жизни
- Поддержка опции "Запомнить меня" с продлённым сроком cookie
- Доступ фильтруется по чатам, назначенным пользователю
- Суперпользователь имеет доступ ко всем чатам

## Фронтенд

Фронтенд собирается из исходников в `src/` через npm-скрипты:

- **TypeScript** — компиляция через `tsc`
- **SCSS** — компиляция через `sass`
- **Tailwind CSS** — обработка через PostCSS

Инструкции по сборке — в [.docs/development.md](../.docs/development.md).
